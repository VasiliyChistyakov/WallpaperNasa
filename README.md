# WallpaperNasa

Приложение использует API [NASA](https://api.nasa.gov). Раз в день по ссылке обновляется картинка, когда пользователь заходит в приложение картинка сохраняется в Фото на устройстве.

- URLSession
- MVVM
- JSON
- UIImage
